My Projects

AR Wall

Click here to view a demonstration video (Please turn on CC). I was the major contributor to this project during my research assistantship at IMAGINE Lab, Center of GIS, Georgia Tech. This project was intended to build an augmented reality application that will assist civil engineering students and also building managers to have a direct preception of in-wall structures. I was in charge of building the 3D model of in-wall structures in 3ds Max and importing into Unity 3D to implement augmented reality function using Qualcomm Vuforia plugins. I also built a simple user interface through C# scripts, which was not demonstrated in this video. The target platform of this application is Android and was successfully built and run on Google Nexus tablet.

Technologies used: 3ds Max, Unity3D, Qualcomm Vuforia, C#
GWCCA Low-Resolution Model

I participated in this project as a member of the IMAGINE Lab, Center of GIS, Georgia Tech. The project was to build low-resolution model for the Georgia World Congress Center Authority (GWCCA) development plan. I was responsible in updating the 3D model of the park area showed in the rendering on the left, including importing tree proxies and building models of foundations and fences in 3ds Max.

Technologies used: 3ds Max
Space Adventure

The project is a cooperative projects with three other people. It is a JavaScript-based web text adventure game. The user would pick a object from the displayed text to interact with. Based on their choices the next paragraph would be different. I was mainly responsible for the back-end functions, including those that would allow user to proceed to the next level of story, change user's statistics based on choices and allow fighting to happen. Link to Page.

Technologies used: HTML, CSS, Bootstrap, Javascript, jQuery
JavaScript Flashcards

The site is a web flashcard sites that I built to help remember JavaScript terms. Users can choose from viewing terms first or viewing definition frist, and they can also choose to have revealed cards stay opened or not. Two convenient buttons are provided for users to close and open all cards at one click. The site only contains front-end logics which was built using JavaScript and jQuery. Link to Page.

Technologies used: HTML, CSS, Bootstrap, Javascript, jQuery
